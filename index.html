AndroSpaceShooter

Ein fesselndes Space-Shooter-Spiel, entwickelt für Android mit Kotlin und Jetpack Compose.

Features

  • Prozedural generierte Grafik: Keine externen Assets, alles wird zur Laufzeit gezeichnet.
  • Performance-Optimierungen: Object Pooling für Partikel, optimiertes Sound-System.
  • Dynamische Soundeffekte: Wabernder Kraken-Loop, spezialisierte Explosionen, Highscore- und Game Over-Sounds.
  • Bonus Space-Krake: Eine Alien-Krake mit Animation und variablem Soundeffekt für Extra-Punkte.
  • Grundlegende Game-Loop, Bildschirm-Schütteln, Schockwelle.
